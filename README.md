# Utils

A collection of lightweight utility scripts for everyday tasks on a bioinformatics workstation.

## Tools

| Script | Description |
|--------|-------------|
| `remove_space_in_filename.py` | Batch-rename files by replacing spaces with underscores. Supports interactive and `-force` modes. |

## Usage

Scripts are standalone Python 3 with no external dependencies:

```bash
python3 <script_name>.py --help
```

## Contributing

New utilities are welcome — keep them self-contained, standard-library-only where possible, and include a `--help` flag.
